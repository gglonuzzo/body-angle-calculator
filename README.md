# Body Angle Calculator

A browser-based biomechanics tool for real-time pose detection and joint angle estimation using MediaPipe.

## How to Use
1. Open the public website:
   https://gglonuzzo.github.io/body-angle-calculator/
2. Click **Start Workout**
3. Allow camera access

All computation occurs locally in the browser. No images or video are stored or transmitted.
